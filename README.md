# python-web-flask-mvc-mysql-simple

## Description
Creates a datatable of `dog` for a flask project.

If path is not found, will default to 404 error.

## Tech stack
- python
  - flask
  - sqlalchemy
- bootstrap
- jquery
- dataTable
- mariadb

## Docker stack
- python:latest
- mariadb:latest

## To run
`sudo ./install.sh -u`
- [web app](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
